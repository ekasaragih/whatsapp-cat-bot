# WhatsApp Cat Bot

## Description

A WhatsApp chatbot using Gemini AI to provide expert information about cats.

## Prerequisites

- Node.js
- Gemini API Key

## Installation

1. Clone the repository
2. Run `npm install`
3. Copy `.env.example` to `.env`
4. Add your Gemini API key
5. Run `node index.js`

## Features

- Cat breed information
- Cat care tips
- Interactive cat-related Q&A
